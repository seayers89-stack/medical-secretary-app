Precise, on purpose

What this is — and isn't

"Verified" can quietly overclaim if used loosely. Here's exactly what the badge means.

This is

  • ✓ A review of a DBS certificate you already hold
  • ✓ ID and reference checks, done by us, for free
  • ✓ Optional — never required to use the platform
  • ✓ Permanently free, with no fee at any stage

This isn't

  • ✕ A new criminal-records search commissioned by Veyn
  • ✕ A skills or competence test (see Skills-Tested, separately)
  • ✕ A requirement to publish or use your profile
  • ✕ A paid service, at this stage or any other

Don't have a DBS certificate yet?

The gov.uk Basic check service is the default route for most secretarial roles (from £21.50). If you're self-employed and need a Standard or Enhanced check, a registered DBS Umbrella Body can process this directly — available since 21 January 2026. Full guidance lives on the Business Hub.
